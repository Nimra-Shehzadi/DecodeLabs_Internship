# GlowNest - Luxury Interior Design Website

## Description

GlowNest is a modern and responsive interior design website developed using HTML, CSS, and JavaScript.

The website showcases luxury interior design services through an elegant user interface, responsive layouts, interactive elements, glassmorphism effects, dark/light theme toggle, animated counters, and modern design aesthetics.

## Features

* Responsive Design
* Modern User Interface
* Dark/Light Theme Toggle
* Animated Statistics Counters
* Glassmorphism Effects
* 3D Hover Animations
* Interactive Navigation Bar
* Contact Form
* Mobile-Friendly Layout

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click the `index.html` file.

OR

Open the folder in VS Code and run using Live Server.
